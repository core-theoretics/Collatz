# Collatz Operator Toolkit

**A synthetic spectral operator for the Collatz (3x+1) conjecture**

**Author:** Jordan Gidman (CoreTheoretics)  
**ORCID:** [0009-0000-2955-5833](https://orcid.org/0009-0000-2955-5833)

## Abstract

Using the Operator Toolkit framework, we constructed a **synthetic Collatz operator** from real trajectories and stopping times.

The operator automatically discovered the correct asymptotic law: stopping time grows as \( c \cdot \ln n \).

**Accuracy on real Collatz data:** **68.0%**

**Core eigenvalue discovered:** λ₁ = 0.23636765
